# Fiesta Salad

![Fiesta Salad][1]{ loading=lazy }

| :timer_clock: Total Time |
|:-----------------------: |
| 0 minutes |

## :salt: Ingredients

- some roasted corn
- some black beans
- some fajita veggies
- some tortilla strips
- some mixed greens
- some chipotle ranch dressing

## :cooking: Cookware

## :pencil: Instructions

### Step 1

Combine roasted corn, black beans, fajita veggies, and tortilla strips over mixed greens.

### Step 2

Top with chipotle ranch dressing.
