# :green_salad: Fiesta Salad

![Fiesta Salad][1]{ loading=lazy }

| :timer_clock: Total Time |
|:-----------------------: |
| 0 minutes |

## :salt: Ingredients

- :bread: some roasted corn
- some [black beans][3]
- :bell_pepper: some fajita veggies
- :bread: some tortilla strips
- :chestnut: some mixed greens
- some [Chipotle Ranch Dressing][2]

## :cooking: Cookware

- :bowl_with_spoon: 1 large bowl

## :pencil: Instructions

### Step 1

Combine roasted corn, [black beans][3], fajita veggies, and tortilla strips over mixed greens.

### Step 2

Top with chipotle ranch dressing.
