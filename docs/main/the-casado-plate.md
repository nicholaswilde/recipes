# The Casado Plate

![The Casado Plate][1]{ loading=lazy }

| :timer_clock: Total Time |
|:-----------------------: |
| 0 minutes |

## :salt: Ingredients

- some jose guevara's gallo pinto
- some hearts of palm ceviche
- some fried green plantains
- some chilero hot sauce
- some nixtamal tortillas
- some corn (optional)
- some sliced avocado (optional)
- some chopped cilantro (optional)
- some lettuce (optional)
- some tomatoes (optional)

## :cooking: Cookware

## :pencil: Instructions

### Step 1

Combine jose guevara's gallo pinto, hearts of palm ceviche, fried green plantains, chilero hot sauce, and nixtamal
tortillas.

### Step 2

Top with corn (optional), sliced avocado (optional), chopped cilantro (optional), lettuce (optional), or tomatoes
(optional).
