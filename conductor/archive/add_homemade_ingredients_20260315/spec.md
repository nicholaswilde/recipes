# Specification: Add Homemade Ingredient Recipes

## Objective
Import recipes for common pantry ingredients and condiments that are frequently used in the project's recipes but are currently treated as store-bought items. This promotes self-sufficiency and expands the `ingredients` collection.

## Scope
The following ingredients have been identified as high-priority candidates for homemade recipes:
1.  **Thai Red Curry Paste**
2.  **Mock Soy Sauce / Tamari Alternative**
3.  **Dijon Mustard**
4.  **Baking Powder**

## Requirements
- Find reliable sources for each ingredient recipe.
- Follow the standard recipe import workflow (`.cook` file creation, image download, `task move` execution, markdown post-processing).
- Categorize these recipes under the `ingredients` or `sauces-and-dressings` categories appropriately.
- Ensure proper emoji usage, tagging, and volumetric-to-weight conversions.
